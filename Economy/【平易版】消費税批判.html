【大学1年生向け】ここがヘンだよ日本の消費税！ PB目標って本当に正しいの？

このノートは、みんなが普段払っている「消費税」や、最近話題の「インボイス制度」、そして国の借金を減らす目標である「プライマリーバランス（PB）黒字化」について、「それって本当に日本のために良いことなの？」と疑問を投げかけるものです。経済の専門家が使う難しい言葉ではなく、できるだけ分かりやすく解説していきます。

1. はじめに：日本の経済、元気ないってホント？

日本の経済は、バブル崩壊後「失われた30年」なんて言われて、長い間あまり元気がない状態が続いています。政府も色々な手は打ってきたんだけど、なかなか良くならない。その間、私たちの生活に身近な税金「消費税」が導入されたり、税率が上がったりしてきました。最近では「インボイス制度」っていう新しいルールも始まりましたね。

このノートでは、 「消費税やインボイス制度、そして国の借金を減らすぞ！っていう目標（PB黒字化目標）って、今の日本経済の仕組み（管理通貨制度）を考えると、ちょっとおかしいんじゃない？むしろ、経済を苦しめてるんじゃない？」 という視点で、問題を掘り下げていきます。

Q1: 「失われた30年」ってよく聞くけど、具体的にどういうこと？
A1: バブル経済が崩壊した1990年代初めから約30年間、日本の経済成長がとても低迷している状態を指します。給料がなかなか上がらなかったり、物価が上がらない（デフレ）状態が続いたりしました。
Q2: PB黒字化目標って何？なんでそんなに重要視されてるの？
A2: PBは「プライマリーバランス」の略で、国の基本的な収入（税収など）と支出（政策に必要な経費。ただし過去の借金の利払い費は除く）のバランスのことです。PB黒字化は、簡単に言うと「借金の利払い以外は、税収だけで国の運営費をまかなえるようにしよう！」という目標です。国の借金が増えすぎると大変だ、という考えから重要視されていますが、このノートではその考え方に疑問を呈します。
Q3: 管理通貨制度って何？
A3: 昔のようにお金（通貨）の価値を金（ゴールド）などで裏付けるのではなく、国（中央銀行）が通貨の発行量を管理・調整する仕組みのことです。日本円もこの制度に基づいています。国は必要に応じてお金を発行できるので、「国の借金＝家庭の借金」とは考え方が全く違う、というのがポイントです。

このノートでは、まず消費税とインボイス制度がどんな仕組みかを見て、それが税金の理想である「公平・中立・簡素」に合っているか考えます。そして、国の借金を減らす目標（PB至上主義）が、今の日本のお金の仕組み（管理通貨制度）とどう違うのか、なぜ問題なのかを説明します。最後に、これらの政策が「失われた30年」にどう影響したのか、そしてもっと良い税金のあり方はないのかを提案します。

2. 消費税とインボイス制度ってどんな仕組み？

2.1 消費税のキホン：モノやサービスにかかる税金

消費税は、商品を買ったりサービスを受けたりするときに、その「価値が付け加えられた部分（付加価値）」に対してかかる税金です。お店（事業者）は、商品を売った時にお客さんから消費税を預かります。でも、そのお店も商品を仕入れたり、お店の電気代を払ったりするときに消費税を払っています。そこで、 預かった消費税から、支払った消費税を差し引いた金額 を国に納める仕組みになっています。

Q4: 「付加価値」って具体的に何？
A4: 例えば、パン屋さんが小麦粉を100円で仕入れて、パンを作って300円で売ったとします。このとき、パン屋さんが生み出した価値は「300円 - 100円 = 200円」です。この200円が付加価値のイメージです。消費税はこの付加価値に対してかかるように設計されています。
Q5: 支払った消費税を差し引く仕組みって、どうしてあるの？
A5: この仕組みを 「仕入税額控除（しいれぜいがくこうじょ）」 と言います。これがないと、原材料→部品→製品→お店→消費者と、モノが動くたびに消費税がどんどん上乗せされて、最終的な値段がすごく高くなってしまいます。それを防ぐために、事業者が払った消費税は引けるようになっているんです。

今の日本では、消費税率は基本的に10%ですが、食べ物（お酒や外食は除く）や新聞（定期購読）は8%の軽減税率になっています。この2種類の税率があることで、計算が少し複雑になっています。

2.2 新ルール「インボイス制度」って何？

2023年10月から始まったインボイス制度は、さっき説明した「仕入税額控除」を正しく行うための新しいルールです。

これからは、事業者が仕入税額控除を受けるためには、原則として 「適格請求書（インボイス）」 という特別な書類を保存しないといけなくなりました。このインボイスには、今までの請求書に加えて、

  • インボイスを発行する事業者の登録番号
  • 税率ごと（10%と8%）の金額と消費税額
などを詳しく書く必要があります。

Q6: インボイスは誰でも発行できるの？
A6: いいえ、税務署に申請して登録を受けた 「適格請求書発行事業者」 だけが発行できます。
Q7: もし取引相手がインボイスを発行してくれなかったらどうなるの？
A7: 原則として、買い手はその取引で支払った消費税を、自分が納める消費税から差し引くこと（仕入税額控除）ができなくなります。つまり、買い手の税負担が増える可能性があるということです。
Q8: 小さなお店やフリーランスはどうなるの？
A8: 年間の売上が1000万円以下の事業者は、消費税の納税が免除される「免税事業者」でした。しかし、免税事業者のままだとインボイスを発行できません。すると、取引相手（特に企業）が仕入税額控除を受けられないので、「インボイスを発行できる課税事業者になってほしい」と言われたり、取引を敬遠されたりする可能性があります。そのため、多くの免税事業者が、インボイス発行のためにあえて課税事業者になるか、取引が減るリスクを受け入れるかの選択を迫られています。ただし、急激な変化を和らげるための経過措置や特例もあります。

2.3 世界の消費税と比べてどうなの？

消費税（VATやGSTと呼ばれることも）は世界中で導入されていますが、日本の制度にはいくつか特徴があります。

  • 税率 : 日本の10%（軽減税率8%）は、ヨーロッパの国々（20%前後が多い）と比べると低いですが、税率が上がった時期が、経済が停滞している時期と重なっているのが日本の特徴です。
  • 免税点 : 日本では年間売上1000万円以下が免税ですが、これは他の国（イギリスは約1500万円相当）と比べてかなり低い水準です。つまり、日本ではより多くの小さな事業者が消費税やインボイス制度の影響を受けやすい構造になっています。
  • インボイスの厳しさ : 多くの国でも仕入税額控除にはインボイスが必要ですが、日本の制度のように「登録しないとインボイスを発行できず」「登録していない事業者からの仕入れは（経過措置後は）全く控除できない」という仕組みは、特に厳しいと言えます。これにより、小さな事業者やフリーランスへの負担が、他国に比べて重くなっています。
日本のインボイス制度は、特に小規模事業者にとって、海外の似たような制度よりも負担が重く、複雑になっている可能性があります。

3. 税金の理想「公平・中立・簡素」からズレてない？

税金には「公平であること」「経済活動を歪めないこと（中立）」「仕組みが分かりやすいこと（簡素）」という３つの理想（租税原則）があります。でも、今の日本の消費税とインボイス制度は、この理想からズレてしまっている、というのがここでの主張です。

3.1 公平なの？ → 低所得者ほど負担が重い！

  • 逆進性 : 消費税は、収入に関係なく同じ率がかかるため、収入に占める生活費の割合が大きい低所得の人ほど、実質的な負担が重くなります。これを逆進性（ぎゃくしんせい）と呼びます。お金持ちも、そうでない人も同じ税率なのは、一見公平に見えても、負担能力を考えると不公平だと言えます。
  • 軽減税率の効果は？ : 食料品などの税率を8%に下げる軽減税率は、この逆進性を和らげるためですが、お金持ちも同じように恩恵を受けるため、本当に困っている人をピンポイントで助けるには非効率的です。
  • インボイス制度の不公平 : インボイス制度は、もともと収入が少ない傾向にある小規模事業者やフリーランスに、事務作業の増加や新たな税負担を強いる可能性があります。これも不公平を大きくする要因です。
Q9: 「逆進性」って、具体的にどういうこと？
A9: 例えば、月収20万円の人と月収100万円の人が、同じように月に10万円分の買い物をしたとします（消費税10%なら税額1万円）。月収20万円の人にとって1万円は収入の5%ですが、月収100万円の人にとっては収入の1%です。収入が低い人ほど、収入に対する税金の割合が大きくなる、これが逆進性です。

3.2 経済活動に影響ないの？(中立) → 大アリ！

  • 消費を冷やす : 消費税は、物やサービスを買う気を削ぐ効果があります。特に、日本のように景気が悪い（デフレ）時に消費税を上げると、ますます人々がお金を使わなくなり、経済全体を冷え込ませてしまいます。
  • おかしな選択 : 店内で食べると10%、持ち帰ると8%のように、似たようなもので税率が違うと、人々は税率が低い方を選びがちになり、本来の経済的な理由とは違うところで行動が歪められます。
  • 取引の歪み : インボイスを発行できない事業者との取引を避ける動きが出ると、自由な競争が妨げられる可能性があります。

3.3 分かりやすいの？(簡素) → 複雑すぎ！

  • 複数税率の面倒さ : 10%と8%の税率が混在することで、お店は売上や仕入れを分けて記録・計算する必要があり、経理作業がとても大変になりました。
  • インボイスの事務負担 : インボイス発行事業者の登録、決められた形式での請求書発行、受け取った請求書のチェックと保存など、特に中小企業にとって、やるべきことが膨大に増えました。
日本の消費税・インボイス制度は、税金の理想である「公平・中立・簡素」のどれにも反してしまっている、という批判です。

4. 国の借金は家庭の借金と同じ？ PB目標 vs. 今の日本のお金の仕組み

日本の財政運営では、長い間「プライマリーバランス（PB）黒字化」、つまり国の借金をこれ以上増やさない（できれば減らす）ことが最重要目標とされてきました。でも、 「そもそも国の借金って、私たちがする借金と本当に同じなの？」 という根本的な疑問があります。

4.1 今の日本のお金の仕組み（管理通貨制度）って？

ここではMMT（現代貨幣理論）という言葉は使いませんが、基本的な考え方を紹介します。

  • 日本は自分でお金（円）を発行できる : 日本政府（と日本銀行）は、自分たちの通貨である「円」を、理論上は必要なだけ発行できます。これは、ドルやユーロで借金している国や、私たち個人の借金とは全く違う点です。
  • 税金の本当の役割 : 国がお金を発行できるなら、税金は何のためにあるのでしょう？
    • 通貨を使わせるため : 税金を円で払う義務があるから、みんな円を使います。
    • 景気の調整弁 : 景気が良すぎて物価が上がりすぎる（インフレ）時に、税金で世の中のお金を吸い上げて、景気を冷ますことができます。
    • 格差の是正 : 所得が多い人から多く税金を取る（累進課税）ことで、貧富の差を和らげます。
    • 特定の行動を促したり抑制したり : 環境に悪いものに税金をかける（環境税）など。
    大事なのは、 政府がお金を使う（支出する）のに、先に税金を集める必要はない ということです。政府がまず支出し、世の中にお金が回り、その一部を後から税金として回収する、という流れです。
  • 国債（国の借金）の正体 : 政府が発行する国債は、政府にとっては「借金」ですが、それを買う国民や銀行にとっては「資産（貯金）」になります。日本政府が円建てで発行している国債は、返せなくなる（デフォルトする）リスクは、経済的な理由ではなく、政治的な理由以外では基本的に考えられません。
  • 本当の限界はモノやサービス : 政府がお金を無限に使えるわけではありません。お金を使いすぎて、世の中のモノやサービス（労働力、資源など）の生産能力を超えてしまうと、激しいインフレが起きてしまいます。これが政府支出の本当の制約です。「国の借金がいくらだから支出できない」のではなく、「インフレにならない範囲で、必要なモノやサービスを供給できるか」が問題なのです。
Q10: 国がお金を発行できるなら、借金は気にしなくていいの？
A10: 「全く気にしなくていい」わけではありませんが、「家庭の借金と同じように、とにかく減らさなきゃ破産する！」と考えるのは間違いだ、ということです。重要なのは借金の額そのものではなく、その借金（国債発行）によって経済がどうなるか（インフレは大丈夫か、経済成長につながるか）を管理することです。
Q11: じゃあ、財務省はどうして「国の借金で財政破綻する！」って言うの？
A11: ここが大きな矛盾点です。財務省は、国内向けには「大変だ！」と危機感を煽って、増税や支出削減（緊縮財政）を進めようとします。しかし、海外の格付け会社（国の信用度を評価する会社）に対しては、「日本が円建ての国債で破綻することは考えられない」と説明しているんです。この態度の違いは、財政破綻論が、緊縮財政を進めるための「方便」として使われている可能性を示唆しています。

4.2 「PB黒字化」って本当に必要なの？

日本のようにデフレ（物が売れず、経済が縮小する状態）から抜け出せない状況で、「PB黒字化」という数字目標のために、消費税を上げたり、公共事業を減らしたりするのは、経済をさらに悪化させるだけではないでしょうか？

PBという数字を良くするために国民の生活が苦しくなり、経済が停滞するなら、それは本末転倒です。PB至上主義は、国民の豊かさよりも、特定の数字目標を優先する考え方であり、その悪影響は計り知れない、というのがここでの批判です。

国の財政を家計簿と同じように考えるのは間違い。PB黒字化目標にこだわりすぎることが、日本経済の停滞を招いている可能性があります。

5. 「失われた30年」と財政政策の関係

5.1 経済停滞の原因は、緊縮財政だった？

日本が30年間も経済停滞から抜け出せないのは、少子高齢化やグローバル化だけが原因ではありません。むしろ、 PB黒字化を目指して繰り返された増税や公共投資の削減といった「緊縮財政」こそが、デフレを長引かせ、経済の足を引っ張ってきた 、と考えるべきです。

特に、消費税率が3%から5%に上がった1997年、5%から8%に上がった2014年、8%から10%に上がった2019年の後は、経済が落ち込んだり、回復が遅れたりしています。これらは景気が悪い時にやるべきこと（財政出動）とは真逆の政策でした。

Q12: 景気が悪い時になぜ緊縮財政をしてはいけないの？
A12: 景気が悪い時は、人々や企業がお金を使わなくなり、物が売れず、さらに景気が悪くなる悪循環（デフレスパイラル）に陥りがちです。こういう時、政府がお金を使う（公共事業を増やす、減税するなど）ことで、世の中のお金の流れを良くし、景気を刺激する必要があります。緊縮財政はその逆で、世の中からお金を吸い上げてしまうので、景気をさらに悪化させてしまうのです。
Q13: PB目標にこだわると、どうして悪循環になるの？
A13: PBを良くしようと緊縮財政をすると、(1)景気が悪くなり、(2)税収が減り、(3)結果的にPBも悪化したり、国の借金の対GDP比が悪化したりすることがあります。(4)それを見て「もっと緊縮が必要だ！」と考えてしまうと、さらに景気が悪くなる…という悪循環に陥ってしまうのです。

5.2 みんなのお財布が寒い！ 消費増税と社会保険料アップ

日本では、消費税が上がる一方で、年金や健康保険などの社会保険料の負担も増え続けています。その結果、働く人たちの手取り収入（可処分所得）や、物価の変動を考慮した実質的な賃金が、この数十年ほとんど増えていないか、むしろ減っています。

手取りが減れば、人々はますます財布の紐を固くします。物が売れなければ、企業も給料を上げたり新しい投資をしたりするのをためらいます。これが日本のデフレと経済停滞の大きな原因の一つです。

5.3 やっぱり消費税が問題？

消費税は、景気が悪い時に上げると、人々の買う力を直接奪ってしまうため、特に悪影響が大きい税金です。本来、政府がお金を使って経済を支えるべき時に、逆のことをしてしまっているのです。

6. 消費税・インボイス制度の問題点 もっと詳しく！

6.1 インボイス制度でツラいのは誰？

インボイス制度で特に大きな影響を受けているのは、これまで消費税の納税を免除されていた小規模事業者やフリーランスです。

  • 事務作業が大変！ : 新しい会計ソフトの導入や、税理士さんへの支払いが増えるなど、コストと手間が増えます。
  • 課税事業者になるか、仕事を失うか？ : インボイスを発行できないと取引先（特に企業）に迷惑がかかるため、「課税事業者になってインボイスを出して」と要求されたり、最悪の場合、取引を打ち切られたりするリスクがあります。課税事業者になると、これまで免除されていた消費税を納める必要が出てきます。
  • 価格交渉力の弱さ : 立場が弱い小規模事業者は、取引先からインボイス発行を理由に値引きを要求されることもあります。

これらの負担は、結果的に所得の低い事業者の生活をさらに苦しくさせ、消費税が持つ「逆進性」をさらに強める効果もあります。新しいビジネスを始めようという意欲も削いでしまいかねません。

Q14: インボイス制度で、実際に困っている人は多いの？
A14: アンケート調査などを見ると、多くの小規模事業者やフリーランスが、事務負担の増加やコスト増、将来への不安を感じていることが分かります。取引先から値引きを要求された、といった声も報告されています。

6.2 食べ物の軽減税率(8%)、本当に良いこと？

食料品などの税率を8%に抑える軽減税率は、一見良いことのように思えますが、これも問題を生んでいます。

  • とにかく複雑！ : 特に飲食店などでは、店内で食べる（10%）のと持ち帰り（8%）で税率が違ったり、仕入れる食材（8%）とお酒や消耗品（10%）で税率が違ったりするため、経理の計算が非常に複雑になりました。
  • どっちの税率？で混乱 : 何が軽減税率の対象になるのか（例えば、おもちゃ付きお菓子は？）、どこからが店内飲食なのか、といった判断基準が難しく、間違いも起こりやすくなっています。
  • 計算ミスが怖い : 複雑さゆえに計算ミスが起こりやすく、後から追加で税金を払うことになるリスクもあります。

6.3 「輸出戻し税」って何？ 大企業だけが得してる？

日本から海外へ輸出される商品には、消費税がかかりません（税率0%）。さらに、その輸出品を作るために国内で仕入れた部品や材料にかかった消費税は、輸出した企業に 還付される（返してもらえる） 仕組みがあります。これを「輸出戻し税」または「輸出還付金」と呼びます。

この制度自体は国際的なルール（消費は最終的に行われる国で課税するという考え）に基づいているのですが、問題はその金額です。トヨタ自動車のような大手の輸出企業には、年間で兆単位の巨額な消費税が還付されています。これは、私たちが納めた消費税全体のかなり大きな部分（約2割とも）を占めています。

Q15: 輸出戻し税にはどんな問題があるの？
A15: いくつか問題点が指摘されています。
  • 実質的な輸出補助金？ : 海外（特にトランプ前米政権）から、「これは輸出企業を不当に優遇する補助金であり、ズルい！」と批判されることがあります。
  • 国内での不公平感 : 消費税は低所得者ほど負担が重いのに、集められた税金のかなりの部分が、儲かっている大企業に戻っている、という構図は、税に対する信頼を損なう可能性があります。
  • 下請けいじめの温床？ : 一部の輸出大企業が、その強い立場を利用して、部品などを納入する下請け企業に対し、「消費税分まけろ」と値引きを強要（買いたたき）している場合があります。この場合、大企業は払っていない（値引かせた）消費税分まで国から還付金として受け取ることになり、実質的な負担は下請け企業に押し付けられてしまいます。

6.4 「消費税は社会保障のため」って本当？

政府はよく「少子高齢化が進む中で、年金や医療などの社会保障制度を維持するために、消費税の増税は仕方ない」と説明します。しかし、この説明には疑問があります。

  • 税金に色はない : 集められた税金は、基本的に一旦国のサイフ（国庫）に入り、予算として様々な用途に振り分けられます。「消費税収が必ず社会保障に使われる」と明確に分けるのは難しいのです（これを「目的税」化の難しさと言います）。
  • 全然足りてない : そもそも、日本の社会保障に必要な費用は、消費税収（10%でも）だけでは全く足りません。消費税はあくまで財源の一部です。
  • 輸出企業への還付 : しかも、集めた消費税の一部は、輸出企業に還付されています。全額が社会保障に使われているわけではありません。
  • お金の問題じゃない？ : そもそも管理通貨制度では、政府の支出能力は税収に縛られません。社会保障に本当に必要なのは、お金そのものよりも、医師や看護師、介護士といった人材や、病院などの設備といった「実物資源」です。「税収が足りないから社会保障を削る」というのは、本末転倒な考え方かもしれません。
「消費税は社会保障のため」という説明は、増税を受け入れてもらうための「口実」で、本当の理由は別にある（例えば、PB黒字化目標の達成や、法人税減税の穴埋めなど）のではないか？ という強い疑念があります。巨額の輸出還付金が存在するのに、「財源が足りない」と言うのは矛盾しているように見えます。

7. もっと良い税金の仕組みはないの？

7.1 憲法と税金：みんなが人間らしく生きるために

日本国憲法第25条には、「すべての国民は、健康で文化的な最低限度の生活を営む権利を有する」と書かれています。国には、国民が人間らしい生活を送れるようにする責任があります。

しかし、低所得者ほど負担が重い消費税に頼りすぎる税制は、この憲法の理念（生存権の保障）に反するのではないか、という問題提起です。税金は、単にお金を集めるだけでなく、 「負担できる人がより多く負担する（応能負担原則）」 ことで、富を再分配し、国民生活を安定させる役割も持つべきです。

7.2 消費税、どう変えるべき？ → 減税か、いっそ廃止？

これまで見てきたように、消費税は経済を冷え込ませ、不公平感も強い税金です。だから、日本経済を元気にして、国民の生活を安定させるためには、 消費税率を大幅に下げる（例えば5%に戻す）か、将来的には廃止する ことを真剣に考えるべきだ、というのがここでの提案です。

Q16: 消費税を減らしたり無くしたりしたら、国の収入が減って大変じゃないの？
A16: その心配はよく分かります。しかし、まず「国の支出は税収に直接縛られない」という管理通貨制度の考え方があります。それに、消費税減税で経済が元気になれば、人々の所得が増え、企業も儲かるようになり、結果的に所得税や法人税といった他の税収が増える可能性もあります。単なる「歳入減」と考えるのではなく、経済を活性化させるための「投資」と捉えるべきだ、という考え方です。

7.3 じゃあ、どんな税金が良いの？

消費税への依存を減らす代わりに、もっと公平で、経済成長とも両立できる税金の仕組みを考える必要があります。ポイントは 「累進性（るいしんせい）」 、つまり所得や資産が多い人ほど、より高い割合の税金を負担する仕組みを強化することです。

  • 所得税 : お金持ち（高額所得者）に対する税率をもっと上げる。株の儲け（金融所得）なども、給料と同じように累進的な税率で課税するように変える。
  • 法人税 : 大企業がうまく税金逃れできないようにルールを整備し、儲けに応じてきちんと税金を払ってもらうようにする。
  • 資産課税 : 相続税や贈与税を見直し、富が一部の人に集中しすぎないようにする。世界で議論されている、超お金持ちの持つ純資産に直接課税する「富裕税」なども検討する価値があります。
  • 簡素化 : 消費税とインボイス制度を無くしたり、単純にしたりすれば、それだけで税金の仕組みは劇的に分かりやすくなります。

8. まとめ：国の財政、考え方を変えよう！

このノートでは、日本の消費税・インボイス制度や、それを正当化してきたPB至上主義が、今の日本のお金の仕組み（管理通貨制度）の現実とズレていることを説明してきました。

財務省は国内では「財政破綻する！」と言いながら、海外には「円建て国債は破綻しない」と説明しています。この矛盾は、PB目標が、経済的な必要性よりも、増税や支出削減を進めるための「考え方（イデオロギー）」であることを示しています。

管理通貨制度のもとでは、日本政府はお金（円）を発行でき、国の支出は税収に直接縛られません。本当の制約は、モノやサービスの供給能力と、それによるインフレのリスクです。

この現実を無視してPB目標にこだわり続けた結果が、「失われた30年」という経済停滞です。特に、負担が不公平で経済を冷やす消費税の導入や増税、そして複雑なインボイス制度は、国民生活を苦しめ、日本経済の足を引っ張ってきました。「社会保障のため」という説明も、巨額の輸出還付金の存在などを考えると、説得力がありません。

今、日本に必要なのは、国の財政に対する考え方を根本から変えることです（パラダイムシフト）。

  1. PB黒字化という目標を捨てる。
  2. 消費税を大幅に減税するか、廃止する。
  3. 負担能力に応じた、公平な税金の仕組み（累進課税中心）を作る。
  4. 政府は、国民の生活向上や経済成長のために、インフレに注意しながら、もっと積極的にお金を使う。

「財政破綻」という架空の危機におびえて、国民生活を犠牲にし続けるのはもうやめるべきです。経済の現実に基づいた、本当に国民のためになる財政政策への転換が必要です。
