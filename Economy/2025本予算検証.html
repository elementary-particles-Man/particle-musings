令和7年度本予算（ 出典 、2025年3月31日成立）を基に、管理通貨制度の視点から分析します。

管理通貨制度に基づく評価

歳出詳細検証

  • 社会保障関係費（37兆2,200億円） : 管理通貨制度下では資金調達に税収依存は不要。高齢化対応として必要だが、インフレ率2-3%下での購買力低下補填が不足。
  • 地方交付税交付金（16兆5,000億円） : 地方経済支援として妥当。国債活用で賄えるため、増額余地あり。
  • 防衛関係費（8兆800億円） : 地政学リスク増大（トランプ関税等）に対応可能。輸入装備コスト増を国債で吸収可。
  • 公共事業関係費（6兆1,200億円） : インフラ投資は経済にプラスの効果をもたらすが、資材高騰で実効性低下リスク。
  • 文教及び科学振興費（5兆5,000億円） : 長期成長に寄与。国債で柔軟に増額可能。
  • 経済協力費（5,500億円） : ODA等の国際的バラマキは効果疑問。国内優先が合理的。
  • 中小企業対策費（1,800億円） : コストプッシュインフレ下で不足。国債で大幅増額が適切。
  • エネルギー対策費（4,200億円） : 燃料高騰対応に不十分。国債活用で補助拡充が必要。
  • 農林水産関係費（2兆2,000億円） : 食料安全保障に寄与するが、インフレ対策不足。
  • その他の経費（6兆5,000億円） : 無駄遣いリスクあり。再配分余地大。
  • 国債費（27兆円） : 日銀保有（600兆円）で実質負担軽減。インフレ4%程度まで耐性あり。

歳出総合評価

管理通貨制度下では、自国通貨建て国債で歳出を賄えるため、現在の114兆3,000億円は柔軟に拡張可能。インフレ率2-3%（上限4%想定）なら、国債費増も吸収でき、経済にプラスの効果を期待できる。しかし、エネルギー対策費や中小企業支援が少なく、コストプッシュインフレへの対応が弱い。経済協力費等の優先度見直しで、国内経済支援を強化すべき。

歳入詳細検証

  • 税収（70兆3,800億円） : 管理通貨制度では税収を財源とする必要なし。消費税23兆円は国民負担増で経済停滞リスク。減税余地あり。
  • 税外収入（5兆8,200億円） : 日銀納付金等は安定収入。増額は困難だが、国債で補完可。
  • 公債金（35兆5,000億円） : 国債依存度31.8%は低め。インフレ率2-3%下で50兆円程度まで増発可能。

歳入総合評価

管理通貨制度を活用すれば、税収依存（62%）を減らし、国債発行を増やすことで歳入を柔軟に拡大できる。現在の35兆5,000億円は保守的すぎ、インフレ調整可能な範囲で増発し、経済にプラスの効果を最大化すべき。税収偏重は国民負担を増やし、成長を阻害するリスクがある。

作成時点で考慮すべきポイント

  • トランプ関税（2025年4月6日開始） : 全輸入品10%、中国34%関税で、日本自動車輸出（170億ドル）に損失リスク。円高（USD-JPY 145.49）で輸入コスト増。
  • 中国報復関税 : レアアース規制等で製造業コスト上昇。
  • エネルギー価格高騰 : 原油・天然ガス価格上昇（インフレ4%想定）が企業・家計を圧迫。
  • 株価変動 : ブラックマンデー（2025年4月7日、S&P 500 32.8%下落、日経平均17.4%下落）で景気後退リスク。

評価：詳細

  • トランプ関税対応 : 防衛費増は妥当だが、自動車産業支援策が不足。中小企業対策費（1,800億円）は不十分。
  • エネルギー対策 : エネルギー対策費（4,200億円）は燃料高騰対応に乏しく、家計・企業支援が欠如。
  • 社会保障費 : 購買力低下への補填策が不足。
  • 公共事業費 : 資材高騰で実効性低下リスク。

評価：総合

予算成立時（2025年3月31日）では、関税やエネルギー高騰が具体化しておらず、対応策が不十分。経済協力費（5,500億円）等の優先度見直しで、景気対策に再配分が必要。インフレ下での柔軟性が欠如し、経済にプラスの効果を十分に発揮できない。

財務省の主張

財務省は「健全財政」を掲げ、国債依存度抑制と増税を推進。税収依存62%を維持し、国債発行を35兆5,000億円に抑え、財政規律を優先。過去の消費税増税（1997年、2014年、2019年）は経済停滞を招いたとの批判を無視。

管理通貨制度における評価

管理通貨制度では税収を財源とせず、国債発行で財政拡大が可能。日銀の国債保有（600兆円）で利払い負担は軽減され、インフレ率4%程度まで財政余地は十分。財務省の「税財源前提」は誤りで、増税不要で経済対策を強化できる。

総括：予算案の問題点と望ましい変更点

問題点

  • インフレ対策不足 : エネルギー対策費（4,200億円）や中小企業支援（1,800億円）が少なく、コストプッシュインフレに対応不足。
  • 家計支援欠如 : 物価上昇分の購買力補填策がなく、消費意欲低下リスク。
  • 経済成長の機会損失 : 緊縮志向が消費を抑制し、経済にプラスの効果を最大化できない。

望ましい変更点

  • エネルギー補助拡充 : 燃料費補助に5兆円規模を追加し、家計負担軽減。
  • 中小企業支援強化 : 原材料高騰対策に3兆円規模の補助金。
  • 国債活用 : 増税せず、国債発行を50兆円程度に増やし、経済にプラスの効果を促進。

補足：日本の通貨制度と財務省ローカルルール

  • 通貨制度 : 日本は管理通貨制を採用。自国通貨建て国債はデフォルトせず、日銀の国債買い支えが可能。インフレ調整が制約。
  • 財務省ローカルルール : 「健全財政」を名目に増税と緊縮を優先。財政危機誇張（国債1,200兆円＝国民1人960万円）や税財源前提は管理通貨制度に反する。
  • 提言 : 財務省は管理通貨制度を活用し、国民負担増ではなく国債による柔軟な財政運営を行うべき。インフレ率4%程度までの財政拡大は経済にプラスの効果をもたらし、成長を加速させる。
